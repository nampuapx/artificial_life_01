Моделирование эволюции с использованием генетического алгоритма by foo52ru
Это последняя версия из тех, что использовались при съемке видео "искусственная жизнь. мир 1".
https://www.youtube.com/watch?v=PCx228KcOow
Код написан для бесплатного интерпретатора Pixilang.
Работает на разных платформах.
Размер экрана 800х480 пиксель, обусловленно размером экрана у планшета, на котором я делал проект.

Нужно запустить pixilang, выбрать файл main и нажать ОК.

Внизу, слева направо:
  текущий сезон
  количество ботов и органики в мире
  прошло тысяч ходов
  прошло ходов
  координаты прокрутки по горизонтали, использовалось в отладке.
  координаты прокрутки по вертикали, использовалось в отладке.
  три ячейки для вывода значения переменных foo0, foo1, foo2, использовалось в отладке.

Справо, сверху вниз
  кнопка стандартного режима отображения
  кнопка режима отображения энергии (краснее == больше энергии)   при zoom24 можно посмотреть числовое значение
  кнопка режима отображения минералов (зеленее == больше энергии) при zoom24 можно посмотреть числовое значение 
  кнопка режима отображения многоклеточных цепочек ( синие, тёмные - крайние в цепочке ) 
  кнопка включения/отключения анимации
  
Отключение анимации увеличивает скорость симуляции.
Кликая по экрану, можно выставить это место по центру экрана.
Проведя по экрану сверху-вниз можно приблизить, а снизу-вверх отдалить.
Существует только три зума.
 
 Для записи видео, нужно в файле main присвоить переменной save_file значение 1
 переменная cadrov указывает, что нужно сохранять кажый 25 кадр.
    save_file = 0       // сохранять ли скрин экрана
    cadrov=25           // сохранять каждый .... кадр 
 При этом в папке с исходниками будут сохраняться скрины в формате PNG
 в стандартном режиме и в режиме отображения энергии.
 Для объединения их в видеофайл, я использовал программу VirtualDub.
  
main     - файл для запуска
constant - собраны константы проекта
bot_class- код команд-генов, которые может выполнить бот
event    - обработка кликов по экрану
paint    - отрисовка экрана
function - отстойник для разных функций, используемых в проекте
описание - читаете сейчас
